# FC Mobile Review Bot

A Discord bot for creating and managing FC Mobile player reviews with backup/restore functionality.

## Features
- 📝 Create player reviews with images and stats
- ✏️ Edit pros, cons, verdict, and alternatives
- 💾 Automatic database storage
- 🔄 Backup and restore system
- 👥 Multi-reviewer support
- 🎮 Persistent buttons (survives bot restarts)
